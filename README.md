# Airflow
Clone from branch of airflow v1-8-stable

## 安装提示
* 执行python setup.py install
* python2.7环境
* mysql 5.7及以上环境，使用MyIASM引擎
* 使用redis 作为celeryExecutor后端
* 使用根目录下面的airflow.cfg作为airflow的配置文件
